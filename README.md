---

## 🎯 Sobre o Projeto

Este projeto consolida análises práticas de dados utilizando as linguagens **R** e **SQL**. O material foi estruturado para demonstrar desde a manipulação básica de dados até a geração de insights estatísticos aplicados a um contexto real (dados hospitalares).
